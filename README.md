Welcome to the Weather Forecasting App! This is a dynamic, client-side web application built using HTML, CSS, and pure JavaScript to fetch and display real-time weather and air pollution data for any city worldwide.

## ‚ú® Features

* **Real-time Weather Data:** Displays current temperature, conditions, humidity, and wind speed.
* **Air Pollution Index (AQI):** Provides the current air quality status for the location.
* **Dynamic UI:** Uses CSS (Bootstrap) and JavaScript for a responsive and engaging user interface, including visual effects using `animate.min.css` and `wow.min.js`.
* **Easy Deployment:** Hosted on Netlify for continuous deployment from the `main` branch.

* **Frontend:** HTML5, CSS3 (with custom styles and Bootstrap)

1.  **Clone the repository:**
    ```bash
    git clone [https://github.com/Satyajeet-0000/Weather-Forcasting.git](https://github.com/Satyajeet-0000/Weather-Forcasting.git)
    ```


*Note: Since this is a client-side application, your API key is likely hardcoded in your JavaScript files (`script_main.js` or similar). For production use, it is recommended to secure your API key using a serverless function (like Netlify Functions).*
